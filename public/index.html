My Week 2 API

Available Endpoints

GET /
Returns a welcome message. curl http://localhost:3000/
POST /user
Create a user greeting. Requires name and email in request body. curl -X POST http://localhost:3000/user \ -H "Content-Type: application/json" \ -d '{"name": "John Doe", "email": "john@example.com"}'
GET /user/:id
Get user profile by ID. Available IDs: 1, 2, 3 curl http://localhost:3000/user/1

Testing Instructions

Using curl: Run the commands shown above in your terminal.

Using Postman: Import the requests and test each endpoint.

Error Testing: Try sending incomplete data or invalid IDs to test error handling.
